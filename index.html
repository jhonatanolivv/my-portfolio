PROJECTS

  • Pipoca Reviews

    Criticine é uma página desenvolvida utilizando HTML, CSS e JavaScript, com o objetivo de apresentar de forma interativa e visualmente atraente informações sobre filmes e críticas cinematográficas. A página é responsiva e se adapta a diferentes dispositivos, proporcionando uma boa experiência de navegação. O projeto utiliza conceitos de design modernos, como tipografia legível, paletas de cores harmoniosas e interações dinâmicas com JavaScript.
